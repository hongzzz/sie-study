## Question Types

### 1. Conceptual Questions
Test understanding of definitions and characteristics.

**Example**:
> Which of the following best describes a debenture?

### 2. Scenario-Based Questions
Test application of knowledge to real-world situations.

**Example**:
> An investor seeking steady income with minimal risk would be best suited for which investment?

### 3. Calculation Questions
Test ability to perform basic calculations.

**Example**:
> A bond with a 5% coupon rate and $1,000 par value pays how much interest annually?

### 4. Comparison Questions
Test ability to distinguish between similar concepts.

**Example**:
> Which statement correctly describes the difference between JTWROS and TIC accounts?

### 5. Regulatory Questions
Test knowledge of laws, rules, and regulations.

**Example**:
> Which act created the SEC?

## How to Use Practice Questions

### 1. Domain-Specific Practice
After completing each domain, practice questions from that domain folder.

**Recommended approach**:
- Study a topic
- Do 10-15 related questions
- Review wrong answers immediately
- Redo wrong questions later

### 2. Mixed Practice
Once you've covered multiple domains, mix questions for better retention.

### 3. Timed Practice
As you get closer to the exam, practice under timed conditions:
- 75 questions in 105 minutes
- Average 1.4 minutes per question

## Tips for Practice Questions

1. **Don't Just Memorize Answers**
   - Understand WHY each answer is correct/incorrect
   - Focus on concepts, not specific questions

2. **Learn from Mistakes**
   - Wrong answers are learning opportunities
   - Keep an error log
   - Identify patterns in mistakes

3. **Time Yourself**
   - Practice under exam conditions
   - Get comfortable with the pace

4. **Review Explanations**
   - Read explanations even for questions you got right
   - May learn additional insights

5. **Redo Difficult Questions**
   - Come back to tough questions after a few days
   - Make sure you truly understand
